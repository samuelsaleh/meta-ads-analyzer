1. LANGUAGE

2. HOOK TYPE
Identifie le type d'accroche principal:
- EMOTIONAL: Joue sur les émotions (peur, désir, fierté, appartenance)
- RATIONAL: Utilise la logique (faits, chiffres, comparaisons, preuves)
- SOCIAL_PROOF: S'appuie sur la preuve sociale (témoignages, popularité, reviews)
- URGENCY: Crée un sentiment d'urgence (temps limité, stock limité, offre spéciale)

3. MARKET STRATEGY
Analyse la stratégie de marché basée sur le contenu et le CTA:
- "Brand Awareness / Storytelling" - Focus sur l'image de marque
- "Product Launch / New Collection" - Lancement de produit
- "Retail / Store Traffic" - Drive-to-store, boutiques physiques
- "E-commerce / Direct Sales" - Vente en ligne directe
- "Lead Generation / Consultation" - Génération de leads, prise de RDV
- "Expansion / New Market" - Expansion géographique
- "Influencer / UGC" - Contenu influenceur ou UGC
- "Retargeting / Loyalty" - Fidélisation, retargeting

4. TARGET AUDIENCE
- Démographie: tranche d'âge, genre, revenus
- Psychographie: valeurs, intérêts, style de vie
- Awareness level: unaware / problem_aware / solution_aware / product_aware
